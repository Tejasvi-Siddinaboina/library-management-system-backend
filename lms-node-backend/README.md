# LibraryMS Backend (Node.js + Express)

## No Java. No Maven. Just Node.js.

## Run
```
npm install
npm run dev
```
Server: http://localhost:8080

## API Endpoints
- POST /api/auth/register
- POST /api/auth/login
- GET  /api/books
- POST /api/books
- PUT  /api/books/:id
- DELETE /api/books/:id
- GET  /api/books/stats
